# Agentic Commerce

AI-powered on-chain commerce system using Arc testnet, USDC, x402-style payments, and Gemini AI.

## Features

- On-chain USDC payments
- Replay-protected receipts
- AI agent spending limits
- x402-style paid API
- Gemini AI product analysis
- Foundry-based smart contracts

## Tech Stack

- Node.js (API server)
- Express
- Foundry (Solidity)
- Arc Testnet
- USDC
- Google Gemini AI
